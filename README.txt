Практическое задание

1. Добавить в систему новый диск, смонтировать, настроить, чтобы автоматически монтировался
 при запуске (используя файл /etc/fstab).
2. (Подразумевается, что используется LVM). Добавить новый диск, добавить его в существующую volume group,
 расширить корневой раздел ОС на новое пространство.
* Добавить на диск произвольный файл, сделать снэпшот с помощью LVM, удалить или изменить файл, 
восстановить файл используя возможности LVM.
