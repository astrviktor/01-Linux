3. Устанавливаем Putty

4. Настраиваем Ubuntu

Обновляемся

	sudo apt-get update
	sudo apt-get upgrade

Устанавливаем и включаем OpenSSH сервер

	sudo apt-cache search ssh | grep open
	sudo apt-get install openssh-server

	sudo systemctl status ssh
	sudo systemctl enable ssh

Устанавливаем mc

	sudo apt-get install mc

Проверяем IP адрес
	
	sudo apt-get install net-tools
	ifconfig | grep inet
